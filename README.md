A color picker (also color chooser or color tool) is a [graphical user interface](https://en.wikipedia.org/wiki/Graphical_user_interface) widget, usually found within graphics software or online, used to select colors and, in some cases, to create [color schemes](https://en.wikipedia.org/wiki/Color_scheme) (the color picker might be more sophisticated than the palette included with the program). Operating systems such as Microsoft Windows or macOS have a system color picker, which can be used by third-party programs (e.g., Adobe Photoshop).
