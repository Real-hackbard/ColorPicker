A color picker (also color chooser or color tool) is a [graphical user interface](https://en.wikipedia.org/wiki/Graphical_user_interface) widget, usually found within graphics software or online, used to select colors and, in some cases, to create [color schemes](https://en.wikipedia.org/wiki/Color_scheme) (the color picker might be more sophisticated than the palette included with the program). Operating systems such as Microsoft Windows or macOS have a system color picker, which can be used by third-party programs (e.g., Adobe Photoshop).

### Websafe Filter:
A color space is a specific organization of colors. In combination with color profiling supported by various physical devices, it supports reproducible representations of color – whether such representation entails an analog or a digital representation. A color space may be arbitrary, i.e. with physically realized colors assigned to a set of physical color [swatches](https://en.wikipedia.org/wiki/Palette_(computing)) with corresponding assigned [color names](https://en.wikipedia.org/wiki/Color_term) (including discrete numbers in – for example – the Pantone collection), or structured with mathematical rigor (as with the NCS System, [Adobe RGB](https://en.wikipedia.org/wiki/Adobe_RGB_color_space) and [sRGB](https://en.wikipedia.org/wiki/SRGB)).

A "color space" is a useful conceptual tool for understanding the color capabilities of a particular device or digital file. When trying to reproduce color on another device, color spaces can show whether shadow/highlight detail and color saturation can be retained, and by how much either will be compromised.
